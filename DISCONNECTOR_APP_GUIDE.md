# Disconnector Mobile App - Quick Start Guide

## 🌐 Network Status Indicator

Look at the top of your app screen:

### 🟢 Online (Green)
- App is connected to the water district server
- Readings submit immediately
- Receipts print in real-time
- **Status**: Everything works normally

### 🔴 Offline (Red)
- App cannot reach the server
- You're in a dead zone with no signal
- Readings are saved locally on your phone
- **Status**: Continue working - readings will sync when online

---

## 📝 How to Submit a Reading

### When Online (Normal)
1. Search for customer
2. Enter meter reading
3. Tap "Submit"
4. Receipt prints/saves
5. ✅ **Done** - Reading sent to server

### When Offline (Dead Zone)
1. Search for customer
2. Enter meter reading
3. Tap "Submit"
4. Message: "Reading saved - will sync when online"
5. Continue to next customer
6. **When you regain signal** → Reading automatically syncs ✅

---

## 🔄 What Happens When Connection Returns

**Automatic Process** (you don't need to do anything):
1. App detects internet connection
2. Status changes to 🟢 Online
3. App automatically sends all saved readings
4. You see confirmation: "X readings synced"
5. Continue working normally

**Time to sync**: Usually 30-60 seconds after getting signal

## ⚠️ Important Notes

### Before You Leave the Office
✅ Make sure app shows **🟢 Online**  
✅ All readings from yesterday are synced  
✅ Check pending count = **0**

### In the Field
✅ Keep WiFi OFF to save battery  
✅ Work normally - app handles offline mode  
✅ Don't force-close the app mid-sync  
✅ Monitor power level - syncing uses battery

### If Something Goes Wrong
❌ App always shows 🔴 Offline?
- Turn phone airplane mode OFF
- Restart app
- Check mobile signal strength
- Contact supervisor

❌ Readings not syncing when back online?
- Wait 60 seconds
- Check if app shows 🟢 Online
- Manually pull down to refresh (if available)
- Restart app if needed

## 📍 Best Practices

### For Efficient Route Work
1. **Morning**: Check 🟢 Online status before leaving
2. **In Field**: Continue work even if offline
3. **Between Routes**: Find a spot with signal to sync
4. **Evening**: Sync before returning to office
5. **Next Day**: Verify all pending = 0 before starting

### To Avoid Data Loss
✅ Submit readings regularly (don't wait until end of day)  
✅ Don't uninstall app with pending readings  
✅ Don't clear app data manually  
✅ Backup phone regularly  
✅ Keep at least 20% battery at all times

### Network Types (Best to Worst)
1. 🟢 **WiFi** at office - Fastest, most reliable
2. 🟢 **LTE/4G** in field - Fast, good coverage
3. 🟡 **3G** in field - Slower, older networks
4. 🔴 **No Signal** - Offline mode (still works!)

## 🆘 Quick Troubleshooting

### "App says offline but I have signal"
```
1. Turn off airplane mode (if on)
2. Toggle WiFi off and back on
3. Wait 10 seconds for app to detect
4. If still offline → restart app
```

### "Reading won't submit"
```
1. Check if status is 🟢 Online
2. If online → Check server connectivity
3. If offline → Submit offline (will sync later)
4. Wait 60 seconds after going online
```

### "App is slow/freezing"
```
1. Close other apps
2. Restart phone
3. Check available storage (need >500MB)
4. Update app if new version available
```

### "Can't find customer"
```
1. Refresh customer list (pull down)
2. Check account number spelling
3. Try searching by name
4. Offline mode? Search works with cached data
```

## 🔄 Sync Status Indicators

### Pending Count Display
- **0 readings pending** - All synced ✅
- **1-5 pending** - Normal, will sync soon
- **5+ pending** - Large queue, find WiFi to sync faster
- **100+ pending** - Check if syncing is stuck

### Sync Indicators in App
- 🔄 **Syncing...** - Currently sending data
- ✅ **Synced** - All readings confirmed on server
- ⚠️ **Failed** - Retry when online, or contact IT

## 💡 Pro Tips

### Maximize Coverage
- Walk to elevated areas for better signal
- Use WiFi when available at customer locations
- Download offline map if doing routes offline

### Efficient Syncing
- Submit readings in batches (don't submit one-by-one)
- Come online between route sections
- Ensure 🟢 Online status before leaving area
- Queue readings overnight, sync in morning

### App Optimization
- Update app regularly (better performance)
- Clear app cache monthly
- Keep phone storage at least 20% free
- Restart app every few days
